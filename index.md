# Privacy Policy for Allergen Hub

Last updated: 2026-02-05

Allergen Hub is an offline-first mobile application designed to help food businesses manage product ingredient and allergen information.

## 1. Data Collection
Allergen Hub does not require user accounts and does not collect personal identifiers such as names, email addresses, or phone numbers.

The app allows users to:
- Capture or select images of product labels and packaging
- Enter product names, ingredients, and allergen information manually

All data entered into the app is stored locally on the user’s device.

## 2. Image Processing
Images captured or selected within the app are used only for:
- Optical Character Recognition (OCR) to extract ingredient text
- Optional visual reference for products and nutrition labels

OCR processing is performed entirely on-device using Google ML Kit.
No images or extracted text are transmitted to external servers.

## 3. Data Storage
All data is stored locally on the device using an embedded database.
Allergen Hub does not use cloud storage, remote servers, or third-party analytics.

## 4. Data Sharing
Allergen Hub does not share data with third parties.

## 5. Data Retention and Deletion
All data remains on the device until the user deletes it or uninstalls the app.
Uninstalling the app removes all stored data.

## 6. Permissions
The app may request access to:
- Camera (to capture product images)
- Photos / media (to select existing images)

These permissions are used solely for core app functionality.

## 7. Children’s Privacy
Allergen Hub is intended for use by food businesses and professionals.
It is not designed for use by children.

## 8. Changes to This Policy
This policy may be updated as the app evolves.
Any changes will be reflected on this page.
